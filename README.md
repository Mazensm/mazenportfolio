# Mazen Portfolio (Vite + React + Tailwind)

## Quick start
```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build
```bash
npm run build
npm run preview
```

## Deploy (GitHub)
```bash
git init
git remote add origin https://github.com/Mazensm/mazenportfolio.git
git add .
git commit -m "Initial commit - portfolio"
git branch -M main
git push -u origin main
```
